# Kişisel Medya Arşiv ve Akış (Streaming) Uygulaması 🎬📚

## 📌 Proje Hakkında
Bu proje; kullanıcıların favori medya içeriklerini tek bir merkezden yönetmelerini, okuma/izleme istatistiklerini düzenli olarak takip etmelerini sağlayan, modern bir UI/UX tasarımına sahip kapsamlı bir mobil uygulamadır.

## 🚀 Temel Özellikler ve Geliştirme Süreci
* **İstatistik ve Streak Takibi:** Kullanıcıların okuma ve izleme alışkanlıklarını analiz ederek "streak" (seri) verilerini tutan algoritmik altyapı.
* **Video Akışı (Streaming) Entegrasyonu:** Dış servislerden ve bulut depolama API'lerinden (Drive API vb.) faydalanarak medya içeriklerinin uygulama içinde kesintisiz oynatılması.
* **Akıllı Önbellekleme (Caching):** `SharedPreferences` kullanılarak API istekleri minimize edilmiş, uygulamanın yükleme süresi ve veri optimizasyonu en üst düzeye çıkarılmıştır.
* **Dinamik Arayüz:** Modern UI/UX prensiplerine uygun, kullanıcı dostu ve akıcı bir Flutter tasarımı.

## 🛠 Kullanılan Teknolojiler
* **Geliştirme Ortamı:** Flutter & Dart
* **Veri Yönetimi:** JSON Encode/Decode, Yerel Önbellekleme (Caching)
* **Ağ (Network):** API Entegrasyonları (REST/GraphQL tabanlı veri çekimi)

Proje dosyaları içerisinde hiçbir telif hakkı barındıran (lisanslı) medya öğesi, manga, novel veya video bulunmamaktadır; sistem tamamen test (mock) verileri üzerinden çalışmaktadır.
